# 🔊 Sound Files

Drop your `.mp3` files here with **exactly** these filenames:

| File | When it plays |
|------|---------------|
| `achievement.mp3` | Achievement toast pops up |
| `task_complete.mp3` | Regular task ticked ✓ |
| `habit_complete.mp3` | Silver 🥈 / Gold 🥇 habit ticked |
| `totem.mp3` | Diamond 💎 / Aujla 🔱 habit ticked (with the explosion) |
| `level_up.mp3` | Level up + confetti 🎉 |
| `bg_music.mp3` | **Loops in background** whenever the app is open |

> Missing files are silently ignored — the app won't break.  
> Use the **🔊 / 🔇** button in the top header to mute/unmute all audio.  
> Mute preference is saved across sessions.
